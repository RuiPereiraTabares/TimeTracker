# ⏰ Control de Horas Extra

Aplicación web sencilla para registrar horas de trabajo y llevar el control de horas extra (sobretiempo).

## ✨ Funciones

- 📝 **Registrar** — Registra tu hora de entrada y salida cada día
- 📅 **Semanal** — Visualiza tus horas extra por día con gráficos de barras
- 📊 **Mensual** — Resumen mensual con desglose por semana (expandible)
- 💾 **CSV** — Exporta/importa tus datos como CSV para respaldo mes a mes

## 🔧 Configuración

- Jornada estándar: **8 horas/día** (40h/semana)
- Todo lo que exceda las 8 horas diarias cuenta como **horas extra**
- Los datos se guardan en el navegador (localStorage)

## 🌐 Uso

Visita: **https://RuiPereiraTabares.github.io/TimeTracker**
